## Featured Projects

<table>
<tr>
<td width="50%" valign="top">

### ResearchMind

A multi-agent research assistant built on **LangChain/LangGraph** — four decoupled agents run in sequence (**Search → Reader → Writer → Critic**) so the Critic reviews only the Writer's output, never its reasoning, keeping feedback unbiased. Context is grounded with **RAG over ChromaDB**, all wrapped in a custom Streamlit interface.

`LangChain` `LangGraph` `RAG` `ChromaDB` `Streamlit` `LLMs`

</td>
<td width="50%" valign="top">

### AG-CT-SGNet — Cotton Leaf Disease Classification

A dual-head **CNN-Transformer** model that diagnoses cotton leaf disease type and severity in one pass. A custom **Dual Spatial-Gated Attention Module (DSGAM)** steers focus onto lesion regions instead of background texture, sharpening accuracy on visually similar disease stages.

`PyTorch` `ResNet50` `Vision Transformer` `Attention Mechanisms`

</td>
</tr>
<tr>
<td width="50%" valign="top">

### HBDRNN — Attention-Augmented Plant Disease Classification

A hybrid deep learning pipeline fusing **CNN (ResNet18)** with **Bi-LSTM and attention** for leaf-image disease classification, combining visual features with geometric descriptors. Training workflows include augmentation, checkpointing, and **Grad-CAM-based interpretability** for transparent inference.

`PyTorch` `ResNet18` `Bi-LSTM` `Attention` `Grad-CAM`

</td>
<td width="50%" valign="top">

### Fake News Intelligent Detection

An NLP classifier benchmarking **SVM, Random Forest, Logistic Regression, Naive Bayes, and Gradient Boosting** on **TF-IDF** features for misinformation detection. A full preprocessing pipeline — tokenization, stopword removal, stemming — feeds the models, deployed via **Flask** for real-time classification.

`NLP` `Scikit-Learn` `TF-IDF` `Flask`

</td>
</tr>
<tr>
<td width="50%" valign="top">

### Acadex360 — AI-Powered Student Collaboration Platform

A scalable full-stack platform streamlining academic collaboration, resource sharing, and performance tracking on a modular **React.js + Node/Express** architecture, with schema-optimized **MongoDB** models and a decoupled system design built to support future AI-driven recommendation engines.

`React.js` `Node.js` `Express.js` `MongoDB` `REST APIs`

</td>
<td width="50%" valign="top">

`Scikit-Learn` `NumPy` `Pandas` `PCA`
